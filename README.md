

# ExpenseTracker

A simple **full-stack expense tracking application** built using **Spring Boot**, **MongoDB**, and **Vanilla JavaScript**.

## Tech Stack

* Spring Boot (Java)
* MongoDB
* HTML, CSS, JavaScript

## Features

* Add, view, update, and delete expenses
* Filter expenses by category
* RESTful API integration

## Run

**Backend:**

```bash
mvn spring-boot:run
```

**Frontend:**
Open `index.html` from the `frontend` folder (backend runs on `localhost:8080`).
